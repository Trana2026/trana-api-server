# AI 자동 기입 기능 국외이전 동의서

*「개인정보 보호법」 제28조의8 및 제28조의9에 따른 별도 동의*

트라나(이하 "회사")는 AI 자동 기입(Auto-Fill) 기능 제공을 위하여 회원이 업로드한 이미지 등을 국외에 소재한 AI 수탁 사업자에게 이전하며, 「개인정보 보호법」 제28조의8에 따라 정보주체의 별도 동의를 받습니다. 아래 내용을 충분히 확인하신 후 동의 여부를 선택하여 주시기 바랍니다.

## 1. 국외 이전의 내용

| 항목 | 내용 |
|---|---|
| **이전받는 자** | OpenAI, L.L.C. |
| **이전받는 자의 연락처** | OpenAI 데이터 처리 부속계약(DPA)에 기재된 개인정보 보호 담당 연락처 |
| **이전되는 국가** | 미국 |
| **이전되는 개인정보 항목** | 회원이 AI 자동 기입 기능에 업로드한 거래 관련 이미지 및 해당 이미지의 분석으로 산출된 자동 기입 결과 |
| **이전 일시 및 방법** | 회원이 AI 자동 기입 기능을 실행하는 시점에, 정보통신망을 통한 API 전송 방식으로 이전 |
| **이용 목적** | 업로드 이미지의 분석을 통한 전자계약서 필드(거래 대상물의 명칭·모델명·상태·거래 금액 등)의 자동 기입 |
| **보유·이용 기간** | • 회사 서버: 자동 기입 처리 완료 즉시 파기<br>• OpenAI: 악용 모니터링 목적으로 최대 30일간 보관한 후 삭제<br>• 단, 자동 기입 결과가 회원에 의하여 채택되어 전자계약서에 반영된 경우, 해당 결과는 전자계약서의 보관기간(체결일과 회원 탈퇴일 중 늦은 날로부터 5년)과 동일한 기간 동안 보관 |

## 2. AI 학습 데이터로의 미사용

① 회사 및 OpenAI는 위 이전된 정보를 AI 모델의 학습 데이터로 이용하지 아니합니다. (OpenAI는 API를 통하여 전송된 데이터를 기본적으로 모델 학습에 사용하지 아니합니다.)

② 다만, 회원이 별도의 동의를 한 경우 또는 해당 정보가 가명처리·익명처리되어 특정 개인을 알아볼 수 없게 된 경우에는 그러하지 아니합니다.

## 3. 동의 거부 권리 및 거부 시 불이익

① 회원은 본 국외 이전에 대한 동의를 거부할 권리가 있습니다.

② 본 동의는 **AI 자동 기입 기능의 이용**을 위한 필수 사항이므로, 동의를 거부하는 경우 AI 자동 기입 기능을 이용할 수 없습니다.

③ 다만, AI 자동 기입 기능은 입력 편의를 위한 부가 기능이므로, 본 동의를 거부하더라도 회원은 전자계약서의 각 필드를 직접 입력하는 방식으로 전자계약 서비스를 이용할 수 있습니다.

## 4. 그 밖의 사항

① 본 동의서에서 정하지 아니한 사항은 회사의 「AI 자동 기입(Auto-Fill) 기능 면책 고지문」, 「개인정보 처리방침」 및 관계 법령에 따릅니다.

② 회원은 본 기능에 고유식별정보·민감정보·금융 보안 정보 등이 노출된 이미지를 업로드하여서는 아니 되며, 이에 관한 사항은 「AI 자동 기입(Auto-Fill) 기능 면책 고지문」 제8조에 따릅니다.
